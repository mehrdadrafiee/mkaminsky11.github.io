Design

I enjoy using "flat" design. I find Google's Material Design language is especially compelling. Boostrap and Polymer are often utilized in my projects, including this web page. Flat designs are uncluttered, easy to read, and easy to use, and have been used extensively by companies like Apple (iOS7) and Google (Android L).

Skills

Swift |
Since I am familiar with scripted languages (such as Javascript), Swift is much easier for me to use that Objective-C. When building apps for iOS or OSX, I almost always use Swift, which is quickly rising in popularity.

Javascript |
Javascript is my greatest strength, and I use it extensively for client-side, server-side, and mobile applications. I run Node.js on my server, and use communicate with client-side code using WebSockets. My Mobile applications are built using Apache Cordova, which allows HTML, Javascript, and CSS to be used to create native iOS and Android applications.

HMTL |
As the backbone of websites, I use HTML as often as I do Javascript. I am also experimenting with Polymer, which allows me to create custom elements, freeing my applications from some of the restrictions imposed by HTML's "default" elements.

CSS |
Flat design, which I mentioned earlier, depends on CSS. Knowing how to properly manipulate CSS is essential to creating responsive, good-looking sites.
Java |
I have taken many courses in Java, including AP Computer Science during the 2013-2014 school year, but do not use it regularly for servers or applications.
mySQL |
I use mySQL to store user preferences in a database, which are then loaded. I connect mySQL to my Node.js server using a module.

Code Your Cloud (web)
This is the project on which most of my time is spent. It is a code editor for Google Drive, available at codeyourcloud.com. It allows for collaborative editing, and incorporates almsot all of the skills I have learned, including mySQL and Node.js. With about 2,000 users on the Chrome Web Store, many people find it to be very useful, and I have used it as a testing ground, experimenting with Polymer and AngularJS, among others.
